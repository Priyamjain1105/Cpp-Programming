# CPP Programming
## How to work
1. Clone
2. For first push `git push -u origin main`
3. To update the changes 
   ```bash
   git add .
   git commit -m "Your message"
   git push
   ```
4. To know the branch
  - `git branch`
5. Link to your remote GitHub repo
  - `git remote add origin <repo-url>`
